# Dataset
This folder contains the scraped data from Kayak website for the period from 2023-03-01 to 2023-05-30 including the following 12 routes:


- AMS ---> NYC
- FCO ---> NYC
- MEX ---> PAR
- NYC ---> AMS
- NYC ---> LUX
- NYC ---> PAR
- PAR ---> NYC
- TLV ---> BKK
- TLV ---> GRU
- TLV ---> MEX
- TLV ---> OSL
- YYZ ---> BKK

This dataset was used to train the models of the project.
